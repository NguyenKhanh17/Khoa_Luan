Introduction to the Project

The data of this project includes information of 100 pigs of the Piétrain NN Français breed, raised at the AXIOM boar testing station in 2020. At this pig farm, an automatic feeding system has been integrated, helping to accurately record the weight and amount of food consumed by each pig each time they visit. This allows detailed data collection on the development of each individual, which can then be analyzed and optimize the breeding process effectively.

Structure of the Database

This database is designed to store and retrieve important information about each individual pig, including:

  • Personal information: Each pig has a unique identification code (ID) to track history and behavior.
  • Age: Information about the age of each pig is calculated in days. This data is important because the nutritional needs and growth rate of pigs vary at each stage of development.
  • Growth parameters: The weight of each pig is recorded at specific times, helping to track the growth process.
  • Daily Feed Intake (DFI): Automatically recorded each time a pig visits the feeding area, helping to determine the exact amount of food that each pig has consumed.
